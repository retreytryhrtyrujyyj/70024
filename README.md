# 70024_基于android的疫情防控志愿服务

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=70024

## 技术介绍
微信小程序;Java;SSM(Spring SpringMVC Mybatis);Java Web;Mysql

## 开发工具
1.微信小程序开发工具 2.IntelliJ IDEA或者Myeclipse 3.Navicat 4.Mysql 5.JDK8

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
